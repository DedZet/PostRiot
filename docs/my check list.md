# Чек-листы тестирования

## Чек-лист тестирования главной страницы

1. Проверка подсветки товара при наведении на него курсором
2. Проверка открытия корзины
3. Проверка перехода к карточке товара

## Чек-лист тестирования карточки товара

1. Проверка выбора размера одежды
2. Проверка добавления товара в корзину
3. Проверка удаления товара из корзины
4. Проверка выхода на главную страницу
5. Проверка перехода к оформлению заказа

## Чек-лист тестирования страницы оформления заказа

1. Проверка ввода ФИО пользователя
2. Проверка ввода электронной почты в соответствующее поле
3. Проверка ввода города пользователя
4. Проверка ввода пункта выдачи товара СДЭК
5. Проверка подтверждения от reCaptcha
6. Проверка кнопки "Оплата" (после того как заполнены все поля и пройдена капча)
7. Проверка оплаты через ЮKassa
