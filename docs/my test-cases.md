# Тест-кейсы

## Главная страница

### Проверка подсветки товара при наведении курсора

**Заголовок:** Подсветка товара при наведении курсора  
**Предусловие:** Открыта главная страница сайта

| Шаг                                              | ОР                                                              |
| ------------------------------------------------ | --------------------------------------------------------------- |
| Навести курсор на карточку товара                | Карточка товара подсвечивается (изменяется фон, тень или рамка) |
| Нажать на иконку корзины в шапке сайта           | Открывается боковая панель или страница корзины                 |
| Нажать на карточку товара или кнопку "Подробнее" | Открывается страница с детальной информацией о товаре           |

## Карточка товара

### Проверка выбора размера одежды

**Заголовок:** Выбор размера одежды в карточке товара  
**Предусловие:** Открыта карточка товара с доступными размерами

| Шаг                                                  | ОР                                                 |
| ---------------------------------------------------- | -------------------------------------------------- |
| Нажать на один из доступных размеров (например, "M") | Размер выделяется (изменяется цвет, рамка или фон) |
| Нажать кнопку "В корзину"                            | Счетчик корзины увеличивается                      |
| Нажать кнопку "Удалить из корзины" (если доступна)   | Товар удаляется из корзины, счетчик обновляется    |
| Нажать на логотип сайта в шапке                      | Открывается главная страница                       |
| Нажать кнопку "Оформить заказ" или "Купить сейчас"   | Открывается страница оформления заказа             |

## Страница оформления заказа

### Проверка ввода полей

**Заголовок:** Оформление заказа
**Предусловие:** Открыта страница оформления заказа

| Шаг                                                  | ОР                                                    |
| ---------------------------------------------------- | ----------------------------------------------------- |
| Ввести корректные ФИО в поле "ФИО"                   | Введенные данные отображаются в поле                  |
| Ввести корректный email в поле "Email"               | Введенный email отображается в поле                   |
| Ввести название города в поле "Город"                | Введенный город отображается в поле                   |
| Выбрать пункт выдачи из выпадающего списка или карты | Пункт выдачи отображается как выбранный<br>           |
| Установить галочку "Я не робот"                      | reCaptcha подтверждается, галочка становится активной |

### Проверка оплаты через ЮKassa

**Заголовок:** Активация кнопки "Оплата" после заполнения формы
**Предусловие:** Все поля заполнены, капча пройдена, нажата кнопка "Оплата",

| Шаг                                         | ОР                                                        |
| ------------------------------------------- | --------------------------------------------------------- |
| Проверить состояние кнопки "Оплата"         | Кнопка активна, доступна для нажатия                      |
| Ввести данные тестовой карты в форме ЮKassa | Отображается процесс оплаты                               |
| Подтвердить оплату                          | Появляется уведомление об успешной оплате, заказ оформлен |
