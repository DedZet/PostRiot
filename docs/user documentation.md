# Пользовательская документация | Post Riot

Данный документ содержит краткое руководство по основному сценарию использования сервиса **Post Riot**.

---

## Основной сценарий покупки

Процесс взаимодействия с магазином от выбора товара до момента оплаты включает следующие шаги:

### 1. Выбор и добавление товара
* **Навигация**: Перейдите в каталог товаров на главной странице `HomePage.jsx`.
* **Просмотр**: Нажмите на карточку товара, чтобы открыть его детальное описание в `ProductPage.jsx`.
* **Параметры**: Выберите необходимый размер из предложенного списка (`sizeList`).
* **Действие**: Нажмите кнопку **«ДОБАВИТЬ В КОРЗИНУ»**, чтобы сохранить товар в `CartContext`.

### 2. Управление корзиной
* **Просмотр**: Откройте боковую панель `CartPanel.jsx`, нажав на иконку корзины в шапке сайта.
* **Проверка**: Убедитесь в правильности выбранных позиций и итоговой суммы. Представляется возможность убрать выбранный товар с помощью кнопки «удалить».
* **Переход**: Нажмите кнопку **«ОФОРМИТЬ ЗАКАЗ»** для перехода на страницу подтверждения данных.

### 3. Оформление заказа (Checkout)
* **Данные доставки**: Введите ваше имя, email, город и адрес пункта выдачи в соответствующие поля формы на `CheckoutPage.jsx`.
* **Безопасность**: Пройдите проверку ReCAPTCHA для подтверждения того, что вы не являетесь роботом.
* **Оплата**: Нажмите кнопку **«ПЕРЕЙТИ К ОПЛАТЕ»**. 
    * *Примечание: Система автоматически создаст заказ в базе данных и перенаправит вас на защищенный шлюз YooKassa*.

---

## Результат
После инициации оплаты ваша корзина в приложении будет очищена, а после успешного завершения транзакции вы попадете на страницу `PaymentSuccess.jsx`.
